# Retail Superstore Sales Analytics (Power BI)

## Project Objective

This project analyzes retail sales data to evaluate overall business performance, product contribution, and customer behavior using an interactive Power BI dashboard.

## Key Analysis Performed

* Sales and Profit KPI tracking
* Regional performance comparison
* Product category contribution
* Top customers analysis
* Customer segment insights
* Profitability analysis

## Tools Used

* Power BI Desktop
* DAX
* Power Query
* Excel/CSV Dataset

## Business Value

The dashboard provides a centralized view of business performance and helps management make data-driven decisions.
