# Customer Profitability Analysis (Power BI)

## Project Objective

The goal of this project is to analyze revenue, cost, and profitability across customers, industries, and business units in order to identify profitable and loss-making segments.

## Key Analysis Performed

* Revenue vs Cost vs Profit comparison
* Gross Margin (GM) and GM%
* Budget vs Actual performance analysis
* Customer ranking by revenue and profitability
* Industry profitability comparison
* Year-to-Date (YTD) and Year-over-Year (YoY) analysis

## Tools Used

* Power BI Desktop
* DAX (Data Analysis Expressions)
* Power Query
* Excel Dataset

## Business Value

This dashboard helps management identify high-value customers, detect loss-making segments, and make data-driven financial decisions.
